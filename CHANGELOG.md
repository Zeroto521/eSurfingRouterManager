# Changelog

> 所有对该项目的显著更改都将记录在此文件中。本项目遵循 [语义化版本控制 (Semantic Versioning)](https://semver.org/lang/zh-CN/)。

## v0.2.0 - 2026-06-23

### Fixed

- 如果数据未加载主动给出提示，避免在数据未加载时，强制修改数据（如切换频率）导致光猫停摆

### Changed

- 在天翼网关登录界面 (`index2.asp`) 隐藏控制台
- 使用天翼官方英文 (eSurfing) 代替原有拼音缩写 (Tianyi)

## v0.1.0 - 2026-06-20

### Added
